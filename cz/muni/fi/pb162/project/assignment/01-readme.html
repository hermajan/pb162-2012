Povinné úkoly:

  1. Vytvořte třídu Vertex2D představující bod v 2D prostoru se souřadnicemi X a Y.
    • Třída bude v balíku cz.muni.fi.pb162.project.geometry.
    • Třída bude mít dva atributy typu double pro hodnoty X a Y + příslušné get a set metody.
    • Metoda String toString() vrátí "[X, Y]", tj. souřadnice bodu v hranatých závorkách.
    • Metoda distance vezme jiný 2D bod jako vstupní parametr a vrátí jeho vzdálenost. Vzdálenost bodů se vypočítá jako

      V Javě se odmocnina vypočítá pomocí volání Math.sqrt().
    • Pozn. pro cvičící: Vysvětlit použití Math.sqrt() s tím, že statické metody budou teprve probrány. Zatím neřešit null hodnoty, bude příště.
  2. Vytvořte třídu Triangle.
    • Třída bude v balíku cz.muni.fi.pb162.project.geometry.
    • Trojúhelník se skládá ze tří vrcholů A, B a C (instance třídy Vertex2D).
    • Metoda void setVertexA(Vertex2D vert) nastaví souřadnice vrcholu A. Podobně přidejte metody setVertexB a setVertexC.
    • Metoda Vertex2D getVertexA() vrátí vrchol A. Podobně přidejte metody getVertexB a getVertexC.
    • Metoda boolean isEquilateral() vrátí true jestliže je trojúhelník rovnostranný. Protože pracujeme s reálnými čísly, nelze jednoduše porovnat délky stran např. pomocí d1 == d2. Je nutné použít test, který bude považovat dvě reálná čísla za shodná, pokud jsou dostatečně podobná např. takto: 
      Math.abs(d1-d2) < 0.001
      kde 0.001 je tolerovaná odchylka.
    • Metoda String toString() vrátí řetězec "Triangle: vertices=[ax, ay] [bx, by] [cx, cy]".
    • Pozn. pro cvičící: Explicitně vysvětlit problém porovnávání reálných čísel a s tím související výraz Math.abs(d1-d2) < 0.001.
  3. Upravte třídu Demo v balíku cz.muni.fi.pb162.project.demo následujícím způsobem:
    • Zrušte výpis textu "Hello World!".
    • Třída vytvoří trojúhelník se souřadnicemi [-1, 0] [0, 1] [1, -1].
    • Na std. výstup vypíše informace o trojúhelníku. Po spuštění by výstup měl vypadat takto: 
           Triangle: vertices=[-1.0, 0.0] [0.0, 1.0] [1.0, -1.0]
    • Pozn. pro cvičící: Vysvětlete resp. pomozte studentům s řádky pro import tříd z jiného balíku.

Nepovinné úkoly:

  1. Upravte třídu Triangle tak, aby se dal trojúhelník rozdělit na čtyři stejně velké části takto (vlevo je původní trojúhelník, vpravo rozdělený):

    Toto rozdělení se používá např. jako základní krok př tvorbě tzv. Sierpinského trojúhelníka. Konkrétně proveďte tyto úpravy:
    • Třída Triangle se bude pomocí atributů odkazovat na tři menší černé trojúhelníky. Na začátku jsou atributy prázdné (rozdělení zatím nebylo provedeno).
    • Metoda boolean divide() vytvoří tři menší trojúhelníky, uloží je do atributů a vrátí true. Pokud již trojúhelník byl rozdělen, metoda neprovede nic a jen vrátí false. Vrcholy menších trojúhelníků jsou vždy v polovině délky stran původního trojúhelníka. Střed hrany (úsečky) má souřadnice S[(x1+x2)/2, (y1+y2)/2], kde [x1, y1] a [x2, y2] jsou vrcholy trojúhelníka.
    • Metoda boolean isDivided() zjistí, jestli již došlo k rozdělení trojúhelníka.
